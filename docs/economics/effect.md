# Экономический эффект внедрения

## Источники эффекта:
- снижение аварийности;
- оптимизация ремонтов;
- сокращение простоев;
- снижение штрафных санкций.

## Оценка:
- снижение CAPEX на 10–15%;
- снижение OPEX на 8–12%;
- сокращение аварийных ремонтов до 30%.

## Срок окупаемости:
1–2 года.
