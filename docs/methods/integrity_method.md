# Методика оценки целостности трубопровода

## 1. Исходные данные
- диаметр трубы;
- толщина стенки;
- параметры дефекта;
- механические свойства материала;
- рабочее давление.

## 2. Расчет по ASME B31G
Допустимое давление определяется по формуле:

Pf = (2 * SMYS * t / D) * (1 - d / t) / M

## 3. Критерии оценки
- коэффициент запаса прочности;
- предельное давление;
- необходимость ремонта.

## 4. Ограничения применимости
Метод применим для коррозионных дефектов
при отсутствии трещин.
